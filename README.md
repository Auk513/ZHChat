##ZHChat
ZHChat是聊天界面的一个库，可以发送语言，图片，文字，表情,位置，视频消息。

![Gif][gif0] 

## [English README LINK]()

##Features
* 界面模仿苹果自带消息应用界面。
* 包含了常用文字，图片，语言，位置，视频消息发送功能。
* 本库聊天界面是基于UIViewController,更容易理解整个UI架构。



## Design Goals
简单方便的集成聊天功能。


## Requirements

* iOS 7.1+
* ARC
